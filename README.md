# pep_pattern.pl
pep_pattern.pl</br></br>
用Perl语言编写了一个pep_pattern.pl脚本,可从一组相关序列中搜索非常相似的序列片段,通过匹配所有可能的氨基酸片段的排列,统计每个匹配模体在序列中的出现频率和位置,搜索蛋白质序列中的2~4个多肽的模体。 </br></br>
